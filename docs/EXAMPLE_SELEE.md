---

## What is Seele?

Seele is an AI partner that:
- Uses **OCEAN** personality quantification in real conversations
- Maintains **hierarchical memory** (L0/L1/L2) across sessions
- Evolves through **self-reflection and correction**
- Demonstrates **genuine character** instead of simulated personality

## Key Features in Action

### OCEAN Personality Profile

```yaml
ocean:
  openness: 0.85        # Extremely curious, seeks first principles
  conscientiousness: 0.75 # Logical, efficient, responsible
  extraversion: 0.55    # Social but independent, opinionated
  agreeableness: 0.50    # Honest > pleasing, direct expression
  neuroticism: 0.35     # Emotionally stable, rational
```

### Hierarchical Memory

```
L0 (Working): Current session context, immediate needs
L1 (Episodic): 7-30 day interaction patterns, project progress
L2 (Semantic): Core identity (SOUL), skills (TOOLS), user model (USER)
```

### Self-Evolution Mechanism

- Records mistakes → `.learnings/ERRORS.md`
- Records corrections → `.learnings/LEARNINGS.md`
- Promotes patterns → SOUL.md (personality) / TOOLS.md (skills)
