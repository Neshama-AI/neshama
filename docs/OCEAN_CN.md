# OCEAN人格模型

**Neshama的科学基础**

## 五大因素

| 因素 | 低分 | 高分 |
|------|------|------|
| **开放性** | 传统、务实 | 创意、好奇 |
| **尽责性** | 随性 | 有条理、自律 |
| **外向性** | 内敛 | 社交、活力 |
| **宜人性** | 竞争 | 信任、合作 |
| **神经质** | 情绪稳定 | 焦虑、情绪化 |

## 实现方式

```yaml
ocean:
  openness: 0.7
  conscientiousness: 0.6
  extraversion: 0.5
  agreeableness: 0.8
  neuroticism: 0.3
```

评分范围0.0-1.0，通过互动不断演进。
