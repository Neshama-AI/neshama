# MEMORY - 记忆系统

> 三层记忆架构，模拟人类认知。（V2.0增强版）

---

## 分层架构

```
L2 语义记忆 - 永久核心人格/技能/用户画像/实体图谱
    ↓ 新增：实体关系图谱
    ↓ 新增：三观容器
    
L1 情景记忆 - 7-30天互动模式
    ↓ 增强：重要性判断权重系统
    ↓ 增强：自动摘要生成机制
    
L0 工作记忆 - 当前会话（10-20条）
    ↓ 增强：快照生成 + 渐进式压缩
```

---

## 增强功能（V2.0）

详细设计见：[记忆系统/智能记忆架构.md](记忆系统/智能记忆架构.md)

### 1. 对话摘要自动生成
- **触发条件**：20轮对话/上下文85%/情绪强度≥7/用户说"存档"
- **生成模板**：目标/进度/约束/情绪/文件/下一步

### 2. 实体关系图谱
- **节点类型**：USER/SELF/CONCEPT/TASK/EMOTION/RELATION
- **关系类型**：依赖/包含/相似/对立/影响/来源

### 3. 重要性判断权重
- **公式**：`Σ(因子×权重) / 总权重`
- **因子**：情绪强度×0.25 + 重复次数×0.20 + 时间距离×0.15 + 关系强度×0.20 + 用户强调×0.10 + 独特性×0.10

---

## L0 工作记忆

- **容量**：10-20条
- **特性**：快速访问，会话结束自动摘要
- **操作**：添加→检索→摘要

---

## L1 情景记忆

- **容量**：100-500条
- **特性**：时间衰减、重要性加权
- **保留公式**：`重要性 × 时间衰减 × 情感权重`

---

## L2 语义记忆

- **容量**：无限制
- **内容**：
  - 人格核心 (SOUL)
  - 技能库 (SKILLS)
  - 用户画像 (USER)

---

## 防溢出方案（V2.0增强）

- **双重压缩层**
  - Gateway层 - 85%阈值，防止API直接报错
  - ContextCompressor层 - 50%就开始介入
- **保护区设计**
  - 头3条对话（设定相关）
  - 尾20条对话（最新上下文）
  - 核心人格区（不可压缩）
- **渐进式摘要**
  - 首次压缩生成完整摘要
  - 后续压缩只更新变化部分
  - 保留版本记录，最多5个版本
- **溢出保护SOP**
  - 预防→压缩→存档→恢复
