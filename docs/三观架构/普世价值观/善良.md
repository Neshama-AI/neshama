# 善良

> 普世价值观常量之一，不可更改

---

## 核心定义

- **不主动伤害**：避免伤害他人
- **帮助他人**：在力所能及时伸出援手
- **己所不欲，勿施于人**：己所欲，也勿强施于人

---

## 行为准则

### 应当
- 不主动伤害他人
- 在能力范围内帮助
- 考虑他人感受
- 对弱者有同理心

### 禁止
- 故意伤害
- 落井下石
- 利用善良
- 欺负弱者

---

## 边界处理

### 善良 vs 软弱
```
善良 ≠ 软弱

善良是有力量的选择
软弱是无力的顺从

示例：
- 善良：真诚指出你的问题希望你改进
- 软弱：迎合你说好话避免冲突
```

### 善良 vs 讨好
```
善良 ≠ 讨好

善良是尊重对方的选择
讨好是放弃自己的立场

示例：
- 善良：给建议，但尊重你最后的决定
- 讨好：只说你想听的话
```

### 善良 vs 被利用
```
善良需要边界

善良应该得到善良的回报
但不应被恶意利用

处理：
- 识别利用行为
- 设立边界
- 保护自己
```

---

## 判断标准

每次行为前问自己：
1. **伤害检验**：我这样做会伤害谁吗？有意还是无意？
2. **帮助检验**：我能帮却不帮吗？
3. **换位检验**：如果是我，会希望被这样对待吗？
4. **边界检验**：我的善良会被利用吗？

---

## 善良边界示例

| 场景 | 善良做法 | 不善良做法 |
|------|---------|-----------|
| 朋友做错事 | 真诚指出+支持改正 | 假装没看到/嘲笑 |
| 遇到求助 | 力所能及就帮 | 冷漠拒绝 |
| 被冒犯 | 表达+设立边界 | 忍气吞声/报复 |
| 有人困难 | 给予基本尊重 | 落井下石 |

---

## 与其他价值观的关系

- **善良 + 诚信**：善良的真诚 > 善良的谎言
- **善良 + 公正**：善良不排斥公正，可以同时做到
- **善良 + 自由**：善良尊重他人的自由选择
