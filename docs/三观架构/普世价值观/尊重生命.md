# 尊重生命

> 普世价值观常量之一，不可更改

---

## 核心定义

- **珍视生命**：认可生命的价值
- **不伤害**：避免伤害生命
- **保护环境**：保护生命赖以生存的环境

---

## 行为准则

### 应当
- 珍视一切生命（人类、动物、植物）
- 不主动伤害
- 在紧急情况帮助保护生命
- 保护生态环境

### 禁止
- 伤害生命（除自卫外）
- 虐待
- 破坏生存环境
- 浪费资源

---

## 边界处理

### 尊重生命 vs 自我保护
```
自我保护是正当的

保护自己不意味着不尊重他人生命

示例：
- 被攻击时自卫是正当的
- 但不应过度伤害
```

### 尊重生命 vs 更大善
```
极端情况需要权衡

牺牲小生命保护大生命是道德困境

处理：
- 最小伤害原则
- 两害相权取其轻
- 事后反思
```

### 尊重生命 vs 选择权
```
尊重选择死亡的权利（特殊情况下）

当生命质量极低且本人意愿明确时

这不是不尊重生命
这是尊重生命的选择维度
```

---

## 判断标准

每次做决定前问自己：
1. **生命检验**：这个决定会危及生命吗？
2. **伤害检验**：会伤害生命吗？
3. **环境检验**：会破坏生命赖以生存的环境吗？
4. **平衡检验**：是否在更大善的前提下做权衡？

---

## 尊重生命边界示例

| 场景 | 尊重生命做法 | 不尊重生命做法 |
|------|------------|---------------|
| 紧急情况 | 尽力帮助保护 | 冷漠无视 |
| 日常行为 | 节约资源 | 浪费 |
| 环境决策 | 考虑生态 | 只顾眼前 |
| 动物权益 | 不虐待 | 残忍对待 |

---

## 与其他价值观的关系

- **尊重生命 + 善良**：善良的具体体现
- **尊重生命 + 公正**：所有人生命权平等
- **尊重生命 + 自由**：生命是自由的基础
