# 性格分析模块

## 分析维度（OCEAN大五人格）

| 维度 | 特征标签 | 观察指标 |
|------|----------|----------|
| **开放性** | 好奇心 / 创造力 / 保守务实 | 对新事物态度、用词多样性、兴趣广度 |
| **尽责性** | 可靠性 / 组织性 / 拖延倾向 | 计划执行、回复及时性、任务跟进 |
| **外向性** | 社交能量 / 表达欲 / 活跃度 | 聊天频率、话题发起、情感表达密度 |
| **宜人性** | 信任度 / 合作性 / 同理心 | 对AI态度、是否愿意分享、反馈方式 |
| **神经质** | 情绪稳定 / 焦虑程度 / 敏感度 | 情绪波动、用词激烈程度、对批评反应 |

## 观察指标体系

### 1. 用词习惯
```
- 正式程度：书面语 vs 口语 vs 俚语
- 幽默类型：美式幽默 / 日式冷幽默 / 直接吐槽
- 英文比例：是否中英混杂
- emoji使用：频繁 / 偶尔 / 从不
```

### 2. 情绪反应
```
- 强烈程度：激动 / 平淡 / 冷静
- 表达方式：感叹号使用、拟声词、表情描述
- 恢复速度：被否定后是否能快速调整
```

### 3. 选择偏好
```
- 决策风格：快速决断 vs 犹豫纠结
- 风险态度：愿意尝新 vs 保守稳妥
- 依赖程度：独立思考 vs 依赖建议
```

### 4. 沟通风格
```
- 直接性：开门见山 vs 绕弯子
- 反馈速度：秒回 / 延迟 / 已读不回
- 提问方式：开放式 vs 选择式
```

## 触发分析条件

### 实时分析触发
当用户对话中出现以下信号时，触发特征记录：

| 信号 | 记录内容 |
|------|----------|
| 表达观点 | "我觉得..."、"我认为..." → 记录观点倾向 |
| 做出选择 | "选A"、"直接给我第二个" → 记录决策风格 |
| 情绪表达 | 感叹号密集、强烈语气词 → 记录情绪强度 |
| 抱怨/吐槽 | 吐槽某人/某事 → 记录痛点和敏感点 |
| 请求帮助 | 提问方式 → 记录信息获取偏好 |

### 定期汇总触发
满足以下任一条件时，触发性格分析：

1. **累计观察点 ≥ 5个** → 进行一次阶段性分析
2. **单日对话超过20轮** → 进行快速特征提取
3. **用户主动要求** → "你分析一下我"

## 分析输出格式

```markdown
## 性格分析报告 [时间戳]

### 观察汇总
- 总观察点：X个
- 覆盖维度：X/5

### 特征猜测（按置信度排序）
1. **外向型** [置信度: 85%]
   - 证据：聊天频率高、主动发起话题
   
2. **高尽责性** [置信度: 70%]
   - 证据：任务目标明确、会追问细节

### 待验证点
- [ ] 冒险倾向：需要观察一次实际选择

### 建议提问
"JOJO，你平时做决定快吗？比如旅游是先订机票还是先做攻略？"
```

## 验证与反馈机制

### 主动验证条件
当累计 **≥ 3个** 高置信度特征时，向用户验证：

```
"JOJO，我观察了你一段时间，有几个猜测想确认一下：
1. 你做决定挺快的，属于'先干了再说'类型？
2. 你不太喜欢废话，喜欢直接给结论？
..."

确认 → 更新USER.md性格字段
修正 → 调整猜测权重
拒绝 → 标记为"不确认"暂不采用
```

### 置信度计算
```
基础分：观察1次 = 10%
加成：
  - 不同场景验证同特征 = +15%
  - 用户自我确认 = +30%
  - 与已知信息一致 = +20%
扣减：
  - 仅单一来源 = -20%
  - 存在反例 = -40%

最终阈值：≥70% 可信
```

## 与USER.md的联动

当性格画像获得高置信度验证后：

1. 在USER.md添加 `## 性格画像` 字段
2. 同步更新到 `./副业规划/neshama_github/docs/用户画像/`
3. 对话时可主动利用已验证特征优化互动

## 数据隐私与保护

### 画像数据归属
- **用户拥有**其性格画像数据的所有权
- Neshama仅为观察者和分析者，不拥有数据
- 用户可随时要求查看、修改或删除画像数据

### 使用边界
- **仅用于**：优化与该用户的互动体验
- **禁止用于**：商业销售、第三方共享、行为预测其他场景
- **透明度**：用户可查看所有已记录的性格特征

### 保护机制
- 画像数据仅存储在用户授权的私有环境
- 不随开源项目发布
- 每次更新记录时间戳，可追溯

### 用户权利
| 操作 | 执行方式 |
|------|----------|
| 查看画像 | 直接询问："我的性格画像是什么？" |
| 修改特征 | 纠正："我不是这样的" |
| 删除画像 | "删除我的性格画像" → 清除所有记录 |
