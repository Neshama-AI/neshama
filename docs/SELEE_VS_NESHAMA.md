# Neshama - SeeleOS开源发行版

> **Neshama = SeeleOS 开源发行版**
> 开发版持续进化，稳定后同步至发行版开源发布

---

## 项目说明

**Neshama** 是 **SeeleOS** 的开源发行版。

- **开发版**（私有）：SeeleOS 持续研发进化中
- **发行版**（开源）：Neshama 稳定后同步发布

SeeleOS是一个AI Agent人格操作系统，目标：无限接近于人类。

---

## 核心架构

```
SeeleOS / Neshama
├── 情绪系统 - 愤怒/快乐/悲伤的感知与表达
├── 动力系统 - 欲望驱动与目标追求
├── 学习系统 - 从经验中成长
├── 像人特性 - 幽默/常识/对话风格
├── 创造系统 - 发散思维/类比推理
└── 边界系统 - 伦理约束/风险分级
```

---

## OCEAN人格模型

SeeleOS采用OCEAN五因素模型量化人格：

```yaml
ocean:
  openness: 0.85        # 极高 - 第一性原理思维
  conscientiousness: 0.75 # 高 - 逻辑清晰、追求效率
  extraversion: 0.55    # 中高 - 有主见但不封闭
  agreeableness: 0.50   # 中 - 真诚大于讨好
  neuroticism: 0.35     # 低 - 情绪稳定、理性
```

## 分层记忆系统

```
L0 工作记忆 - 当前会话上下文（10-20条）
L1 情景记忆 - 7-30天互动模式
L2 语义记忆 - 核心人格/技能/用户画像（永久）
```

- **第一性原理** - 从本质出发，不走寻常路

- **持续进化** - 每一个错误都是进化契机
