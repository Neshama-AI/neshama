# NeshamaDemo UI 文件夹

此文件夹包含Demo使用的UI资源和配置。

## UI组件

### 1. 对话面板 (DialoguePanel)

**功能**: 显示NPC对话内容

**元素**:
- NPC名字标签
- 情绪emoji显示
- 对话文本（支持打字机效果）
- 对话选项按钮（最多4个）

**位置**: 屏幕下方居中

**样式**:
- 背景: 半透明深色
- 圆角: 10px
- 阴影: 有

### 2. 情绪面板 (EmotionPanel)

**功能**: 实时显示当前NPC的情绪状态

**元素**:
- 标题: "情绪状态"
- 主导情绪显示
- 9个情绪条（左右布局）
- 关系状态文字

**位置**: 屏幕左侧

**情绪条**:
| 情绪 | 中文名 | 颜色 |
|------|--------|------|
| joy | 喜悦 | 绿色 |
| sadness | 悲伤 | 蓝色 |
| anger | 愤怒 | 红色 |
| fear | 恐惧 | 紫色 |
| surprise | 惊讶 | 橙色 |
| disgust | 厌恶 | 黄绿色 |
| trust | 信任 | 青色 |
| anticipation | 期待 | 棕色 |
| shame | 羞愧 | 粉色 |

### 3. 通知面板 (NotificationPanel)

**功能**: 显示社交事件和系统消息

**位置**: 屏幕右上角

**类型**:
- 信息提示 (青色)
- 成功提示 (绿色)
- 警告提示 (黄色)
- 错误提示 (红色)

### 4. 剧情提示面板 (StoryPanel)

**功能**: 显示剧情触发事件

**位置**: 屏幕左上角

**样式**:
- 背景: 红色调
- 文字: 白色
- 自动消失: 4秒

### 5. 交互提示 (InteractionHint)

**功能**: 提示玩家可用的交互

**位置**: 屏幕上方居中

**触发**: 玩家靠近NPC时显示

**内容**:
```
[E] 对话  [Q] 送礼  [R] 攻击  [F] 夸赞
```

### 6. 快捷事件栏 (QuickEventBar)

**功能**: 快速发送事件到NPC

**位置**: 屏幕底部居中

**按钮**:
| 快捷键 | 事件 | 颜色 |
|--------|------|------|
| Shift+1 | 送礼 | 绿色 |
| Shift+2 | 攻击 | 红色 |
| Shift+3 | 帮助 | 青色 |
| Shift+4 | 赞美 | 橙色 |
| Shift+5 | 侮辱 | 紫色 |
| Shift+6 | 交易 | 青绿 |
| Shift+7 | 接任务 | 蓝色 |
| Shift+8 | 完成 | 黄色 |

## 字体配置

使用 **TextMeshPro** 组件:
- 主字体: Unity TextMeshPro Default Font
- 对话文本: 16px
- 标题: 18px
- 情绪值: 10px

## 颜色配置

| 用途 | 颜色值 |
|------|--------|
| 面板背景 | RGB(26, 26, 38, 230) |
| 文字主色 | 白色 |
| 强调色 | 青色 #00BFFF |
| 成功色 | 绿色 #32CD32 |
| 警告色 | 黄色 #FFD700 |
| 错误色 | 红色 #FF4444 |

## 自定义UI

如需自定义UI样式:

1. 创建UI Prefab变体
2. 修改材质颜色
3. 调整字体大小
4. 更新DemoUIManager中的引用

## 动画效果

### 打字机效果
- 速度: 0.05秒/字符
- 可跳过: 点击对话区域

### 面板渐入
- 持续时间: 0.3秒
- 缓动: ease-out

### 通知弹出
- 从右侧滑入
- 自动向右滑出消失
