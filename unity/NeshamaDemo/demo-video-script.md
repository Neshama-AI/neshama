## 4:00 - 4:30 | 代码展示

**代码显示**:
```csharp
// 1. 添加组件
public class MyNPC : MonoBehaviour
{
    public NPCSoul npcSoul;
    
    void Start()
    {
        // 2. 发送事件
        npcSoul.SendEvent(GameEventType.player_attacked, 1.0f);
        
        // 3. 监听情绪变化
        npcSoul.OnEmotionChanged += OnEmotionChanged;
        
        // 4. 获取对话响应
        var response = await npcSoul.ChatAsync(playerMessage);
        
        // 5. 显示结果
        ShowDialogue(response.Text);
    }
}
```

**旁白**:

**画面**: 快速闪过更多API

## 视频制作指南

### 配音建议
- 语速: 中等偏慢，让观众跟上
- 语气: 专业但有激情
- 旁白长度: 约600字

### 画面要求
- 分辨率: 1080p minimum, 4K preferred
- 帧率: 60fps
- 颜色: 暖色调酒馆氛围

### 剪辑节奏
- 0:00-0:15: 神秘感铺垫
- 0:15-0:45: 场景展示
- 0:45-2:00: 核心功能演示
- 2:00-3:30: 深度功能演示
- 3:30-4:00: 对比冲击
- 4:00-4:30: 技术说服
- 4:30-5:00: 号召行动

### BGM建议
- 开场: 神秘/史诗
- 场景展示: 轻快的冒险音乐
- 演示环节: 无BGM，突出对话
- 结尾: 积极向上的结尾曲

### 字幕要求
- 中文为主
- 关键术语双语显示
- 情绪值等数据高亮显示
