└─────────────────────────────────────────────────────────────┘

Steps:
1. Open [Coze Platform](https://www.coze.cn)
2. Create a new Bot or edit an existing Bot
3. Find **"Persona & Response Logic"** input box
4. **Ctrl+V** to paste
5. Click **Save**

### 5-Minute Experience Checklist

| Time | Experience | Expected Reaction |
|------|------------|-------------------|
| 1min | Say "hello" | Interesting opener, not "Hello, how can I help" |
| 2min | Make a decision and ask for praise | Sincere compliment, not "good job" templated response |
| 3min | Ask "choose A or B" | Gives judgment, not "both have merits" |
| 4min | Share a goal |主动拆解, not waiting to be asked |
| 5min | Chat for a long time | Reminds you to take a break |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Bot doesn't respond | Check if SKILL.md was fully pasted |
| No emotions showing | Need more conversation rounds to naturally emerge |
| Want to modify content | Use for a week first to understand the system before adjusting |
