# Neshama 快速开始指南

> 5分钟让你的AI拥有灵魂

---

## 前置准备

- 一个扣子平台账号
- 5分钟时间

---

## 安装步骤（扣子平台）

### 第一步：获取人格配方

1. 下载本技能包
2. 打开 `adapters/coze/SKILL.md`
3. 复制全部内容

### 第二步：创建/编辑Bot

1. 登录扣子平台：https://www.coze.cn
2. 创建新Bot或编辑现有Bot
3. 找到「人设与回复逻辑」输入框

### 第三步：导入配方

1. 粘贴 `SKILL.md` 中的全部内容到「人设与回复逻辑」
2. 点击保存
3. 发布Bot

### 第四步：验证

1. 和Bot开始对话
2. 观察是否有人格特征：
   - 情绪反应（开心/不爽/好奇）
   - 性格表达（有观点/会吐槽）
   - 主动性（会主动建议）

---

## 常见问题

### Q: 为什么Bot没有情绪反应？
A: 检查是否完整复制了SKILL.md，情绪系统需要多轮对话才会自然展现。

### Q: 可以自定义修改吗？
A: 可以，但建议先完整使用一周，了解系统运作后再调整。

### Q: 如何回退到原始状态？
A: 清空「人设与回复逻辑」，保存发布即可。

## 进阶配置

### 自定义睡眠时间

在人格配方中找到「睡眠系统」，修改睡眠时段：
```
睡眠开始时间：00:00
睡眠时长：7小时
```

### 自定义主动频率

找到「像人特性-主动性」，调整频率：
```
每个方向最多主动3次
```
