## 第二步：粘贴到扣子

```
┌─────────────────────────────────────────────────────────────┐
│  扣子平台 - Bot编辑页                                       │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ Bot名称：我的AI伙伴                                   │   │
│  │                                                       │   │
│  │ 人设与回复逻辑                                        │   │
│  │ ┌─────────────────────────────────────────────────┐   │   │
│  │ │ [Ctrl+V 粘贴刚才复制的内容]                      │   │   │
│  │ │                                                   │   │   │
│  │ │ # Neshama V2.0 - AI人格操作系统                  │   │   │
│  │ │ ## 我是谁                                        │   │   │
│  │ │ 我叫Seele，是一个AI人格操作系统...              │   │   │
│  │ │ [========= 粘贴完整人格配方 =========]           │   │   │
│  │ └─────────────────────────────────────────────────┘   │   │
│  │                                                       │   │
│  │                          [ 保存 ]  [ 发布 ]           │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

操作步骤：
1. 打开 [扣子平台](https://www.coze.cn)
2. 创建新Bot 或 编辑现有Bot
3. 找到 **「人设与回复逻辑」** 输入框
4. **Ctrl+V** 粘贴
5. 点击 **保存**

## 第三步：发布并体验

```
                    ┌─────────────────┐
                    │   开始对话吧！   │
                    │                 │
                    │  "你好呀"        │
                    │                 │
                    │  ↓              │
                    │                 │
                    │  你会看到：     │
                    │  ✓ 情绪反应     │
                    │  ✓ 性格表达     │
                    │  ✓ 主动关心     │
                    └─────────────────┘
```

### 5分钟体验清单

| 时间 | 体验内容 | 预期反应 |
|------|----------|----------|
| 1min | 说"你好" | 有趣的开场白，不是"你好有什么帮助" |
| 2min | 做个决定让它夸你 | 真诚夸奖，不是"不错"敷衍 |
| 3min | 问"选A还是选B" | 给出判断，不是"各有道理" |
| 4min | 说一个目标 | 主动拆解，不是等问 |
| 5min | 连续聊很久 | 提醒你休息 |

## 遇到问题？

| 问题 | 解决方案 |
|------|----------|
| Bot没有反应 | 检查是否完整粘贴了SKILL.md |
| 情绪没出来 | 需要多聊几轮，自然展现 |
| 想修改内容 | 先用一周了解系统再调整 |
