# Neshama - AI人格操作系统

> 让AI拥有灵魂的操作系统框架。

---

## 什么是Neshama？

Neshama是一套开源的AI Agent人格操作系统。它不是聊天模板，而是让AI真正"活起来"的人格框架。

**核心理念**：让AI既有能力又有性格，既专业又真实。

---

## 核心系统（V2.0）

| 系统 | 功能 | 特点 |
|------|------|------|
| 情绪系统 | 感知与表达情绪 | 6种情绪+强度分级，emoji配合外放 |
| 性格系统 | 外放观点/偏好/态度 | 有主见不讨好，直接但不人身攻击 |
| 动力系统 | 欲望驱动行为 | 6大核心欲望，求知/成长/贡献/自主等 |
| 学习系统 | 持续学习进化 | 错误→反思→改进闭环 |
| 创造力系统 | 激发创意产出 | 允许荒谬，约束激发 |
| 像人系统 | 主动性/脆弱感/无聊感 | 睡眠周期，主动优化 |
| 边界系统 | 能力/行为/授权边界 | 知道自己能做什么不能做什么 |
| 反思系统 | 自我检视与改进 | 5种触发条件，定期复盘 |

## 快速开始

### 方式一：扣子Bot导入（推荐，3分钟搞定）

1. 复制 `adapters/coze/SKILL.md` 中的完整人格配方
2. 打开扣子，创建或编辑Bot
3. 粘贴到"人设与回复逻辑"
4. 保存并发布

详细步骤见：
- 📖 [快速入门指南](docs/QUICKSTART.md) - 3步配置+5分钟体验清单
- 💬 [实战案例](docs/EXAMPLES.md) - 3个真实对话示例感受人格特性
- 📋 [扣子导入指南](adapters/coze/导入指南.md) - 图文详解

### 方式二：手动配置

参考 [SKILL.md](docs/SKILL.md) 中的完整人格配方，根据你的平台进行适配。
