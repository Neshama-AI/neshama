# OpenClaw Adapter

**Neshama Adapter for OpenClaw**

## Overview

OpenClaw is an open-source AI Agent framework. OpenClaw Adapter enables Neshama framework on OpenClaw.

## Usage

### 1. Get Config

Import in OpenClaw persona:

```
【Persona Config】
OCEAN Profile: Openness 0.7/Conscientiousness 0.6/Extraversion 0.5/Agreeableness 0.8/Neuroticism 0.3
```

### 2. Memory System

```
【Memory Rules】
- L0 Working: Current session
- L1 Episodic: 7-30 days
- L2 Semantic: Permanent
```
