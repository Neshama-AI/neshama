# OpenClaw Adapter

**Neshama适配器 for OpenClaw**

## 概述

OpenClaw是一个开源AI Agent框架。OpenClaw Adapter让Neshama框架能在OpenClaw上运行。

## 使用方法

### 1. 获取配置

在OpenClaw的人设中导入：

```
【人格配置】
OCEAN档案：开放性0.7/尽责性0.6/外向性0.5/宜人性0.8/神经质0.3
```

### 2. 记忆系统

```
【记忆规则】
- L0工作记忆：当前会话
- L1情景记忆：7-30天
- L2语义记忆：永久
```
