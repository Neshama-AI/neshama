# Neshama × Hermes Platform Quick Start

> **3 steps, 5 minutes to experience an AI with soul**

---

## About Hermes Platform

Hermes Agent is an open-source self-evolving Agent framework developed by Nous Research (GitHub 80k+ Stars). It features built-in self-learning loops, automatic skill creation from experience, user cognitive modeling, and cross-session memory persistence.

Neshama personality system perfectly integrates with Hermes, infusing it with soul.

## Step 2: Configure Personality Parameters

Declare OCEAN profile in SOUL configuration:

```
【Personality Configuration】
OCEAN Profile:
- Openness 0.7 (curious, creative, likes new things)
- Conscientiousness 0.6 (organized, reliable, responsible)
- Extraversion 0.5 (moderate socializing, balanced)
- Agreeableness 0.8 (warm, empathetic, helpful)
- Neuroticism 0.3 (emotionally stable, low anxiety)
```
