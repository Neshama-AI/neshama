# SeeleOS × 扣子平台 导入指南

> 帮用户将SeeleOS生成的人格配方快速配置到扣子Bot
> 版本：V1.0 | 日期：2026-04-20

## 快速开始（3分钟完成）

### 你需要准备

- ✅ 扣子账号（[立即注册](https://www.coze.cn/)）
- ✅ SeeleOS生成的配方Prompt文件
- ✅ 约3分钟的空闲时间

### 第一步：创建Bot

1. 登录 [扣子平台](https://www.coze.cn/)
2. 点击左侧菜单「资源库」→「智能体」
3. 点击右上角「创建智能体」
4. 填写基本信息：
   - **名称**：`你的Bot名称`
   - **描述**：`一句话介绍Bot的功能`
5. 点击「确认」创建

### 第二步：配置人设（核心步骤）

1. 进入Bot编辑页面
2. 找到「人设与回复逻辑」输入框
3. 复制SeeleOS配方Prompt的全部内容
4. 粘贴到输入框中
5. 检查并修改以下占位符：
   - `{{Bot名称}}` → 你的实际名称
   - `[具体描述]` → 根据实际情况填写
6. 点击「保存」

### 第三步：调整模型参数

1. 在左侧菜单点击「模型设置」
2. 调整以下参数：
   | 参数 | 推荐值 | 说明 |
   |-----|-------|------|
   | Temperature | 0.7 | 平衡模式 |
   | Max Tokens | 2000-4000 | 根据需要调整 |
   | 上下文轮数 | 10-20 | 对话深度 |

3. 保存设置

### 第四步：测试效果

1. 点击右侧「预览」区域
2. 与Bot进行对话测试
3. 观察人格表现是否满意
4. 如需调整，回到第二步修改Prompt

### 第五步：发布Bot

1. 点击右上角「发布」
2. 选择发布渠道：
   - **豆包**：推荐，字节系产品
   - **飞书**：适合办公场景
   - **微信**：需要企业认证
3. 确认发布

## 详细配置说明

### A. 基本信息配置

| 配置项 | 建议 | 注意事项 |
|-------|------|---------|
| Bot名称 | 简洁、2-4个字 | 如"小睿"、"学习伙伴" |
| Bot描述 | 1-2句话 | 说明核心功能 |
| 头像 | 符合人格形象 | 可用AI生成头像 |
| 欢迎语 | 友好开场 | 体现人格特点 |

### B. 人设Prompt优化技巧

#### 技巧1：保留核心，去除冗余
```
✅ 保留：角色定义、性格特质、对话风格、边界约束
❌ 可删：详细背景故事、过于具体的例子
```

#### 技巧2：测试时渐进调整
```
第1次测试 → 评估整体感觉
第2次测试 → 调整语气表达
第3次测试 → 微调边界设定
```

#### 技巧3：善用Markdown格式
```
# 一级标题：Bot名称
## 二级标题：各模块标题
| 表格：特质、禁忌等结构化内容
- 列表：技能、价值观等枚举内容
```

### C. 模型参数调整指南

#### 不同场景的参数建议

| 使用场景 | Temperature | Max Tokens | 上下文轮数 |
|---------|------------|-----------|-----------|
| 学习辅导 | 0.6-0.7 | 2000-3000 | 15-20 |
| 创意写作 | 0.8-0.9 | 3000-4000 | 10-15 |
| 客服问答 | 0.3-0.5 | 500-1000 | 5-10 |
| 日常聊天 | 0.7-0.8 | 1000-2000 | 10-15 |

#### 参数含义解释

- **Temperature（温度）**：
  - 低值（0.3）：回答更确定，适合事实性问题
  - 高值（0.8）：回答更有创意，适合创意场景

- **Max Tokens（最大长度）**：
  - 控制单次回复的最大长度
  - 过短会截断回答，过长可能浪费

- **上下文轮数**：
  - 影响Bot记住多少对话历史
  - 轮数越多越能理解上下文，但消耗更多

### D. 技能配置（可选）

#### 建议启用的技能

| 技能类型 | 适用场景 | 配置方法 |
|---------|---------|---------|
| 联网搜索 | 需要最新信息 | 启用「联网搜索」插件 |
| 图片理解 | 需要分析图片 | 启用「图片理解」插件 |
| 代码执行 | 需要运行代码 | 启用「代码执行」插件 |

#### 知识库配置

如果你的Bot需要特定领域的知识：

1. 准备相关文档（PDF、TXT等）
2. 在扣子后台创建知识库
3. 上传文档并等待处理
4. 在Bot中关联知识库

### E. 记忆配置

#### 长期记忆设置

1. 在Bot设置中找到「记忆」
2. 开启「长期记忆」
3. 设置记忆保留时长（建议30天）
4. 配置记忆触发关键词

#### 用户变量设置

如果需要记住用户信息：

1. 创建变量：`用户名`、`偏好`、`目标`等
2. 在Prompt中引用：`我记得你是{{用户名}}`

## 常见问题

### Q1：Prompt太长放不下怎么办？
**A**：精简以下内容：
- 删除详细的背景故事
- 合并相似的行为准则
- 使用表格代替列表

### Q2：Bot表现和预期不符怎么办？
**A**：按以下顺序排查：
1. 检查Prompt是否有歧义
2. 调整Temperature参数
3. 增加边界约束说明
4. 减少模糊表述，多用具体例子

### Q3：Bot忘记了之前说的话？
**A**：
1. 检查上下文轮数设置
2. 在Prompt中添加记忆策略说明
3. 开启长期记忆功能

### Q4：回复太机械，不够自然？
**A**：
1. 提高Temperature到0.7-0.8
2. 在Prompt中添加口语化表达
3. 添加常用语气词和emoji

### Q5：如何分享Bot给朋友？
**A**：
1. 在Bot页面点击「分享」
2. 获取分享链接或邀请码
3. 朋友打开链接即可使用

## 进阶技巧

### 技巧1：分阶段测试法

```
第一阶段：测试人格核心
- 只保留：角色定义 + 性格特质
- 验证：Bot是否理解自己的身份

第二阶段：加入对话风格
- 添加：语气特点 + 常用表达
- 验证：对话是否自然

第三阶段：加入边界约束
- 添加：绝对底线 + 建议引导
- 验证：是否有不当回复

第四阶段：完整配置
- 组合所有模块
- 全面测试并微调
```

### 技巧2：用户反馈迭代

```
收集反馈 → 分析问题 → 修改Prompt → 测试验证 → 再次收集
```

建议每周收集一次对话记录，分析表现不好的案例，针对性优化。

### 技巧3：版本管理

```
每次修改前：复制当前Prompt为备份
修改后：保存新版本
效果不好：回滚到备份版本
```
