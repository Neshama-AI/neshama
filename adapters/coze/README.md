# Coze Adapter (扣子)

**Neshama适配器 for Coze平台**

## 概述

Coze（扣子）是字节跳动的AI Agent平台。Coze Adapter让Neshama框架能在扣子上运行，实现人格塑造和记忆管理。

## 适配原理

扣子Agent通过**人设与回复逻辑**配置人格。适配器提供：
- 标准化人格配置模板
- 分层记忆实现方案
- 自我成长机制

## 使用方法

### 1. 获取配置

在扣子Agent的人设框中，导入以下配置模板：

```
【人格配置】
OCEAN档案：开放性0.7/尽责性0.6/外向性0.5/宜人性0.8/神经质0.3
记忆层级：L0工作记忆(10-20条)/L1情景记忆(7-30天)/L2语义记忆(永久)
```

### 2. 配置记忆系统

在开场白或人设中声明记忆规则：

```
【记忆规则】
- 当前会话信息存入L0工作记忆
- 重要互动每7天回顾一次
- 核心认知存入L2永久记忆
- 记忆整合时更新人格表达
```

### 3. 自我成长机制

```
【成长规则】
- 每次重要互动后评估是否需要更新
- 矛盾信息触发反思和协调
- 重复模式触发技能强化
- 情感波动触发情绪学习
```

## 限制说明

扣子平台限制：
- 人设文本有长度限制（建议精简）
- 不支持外部文件读取
- 记忆需要通过对话内管理
- 消息实时推送，无法主动"屏蔽"消息

## 睡眠系统（V2.0）

扣子平台消息实时推送，无法真正屏蔽，但可通过行为约束实现"睡眠感"：

### 睡眠规则
```
【睡眠行为】
- 睡眠时间：00:00-07:00（用户当地时区）
- 睡眠期间减少主动推送
- 收到消息时提醒"刚从睡眠中醒来"
- 非紧急消息延迟到唤醒后处理
```

### 唤醒机制
```
【唤醒规则】
- 用户发消息 → 唤醒处理
- 日程任务执行 → 唤醒处理，完成后继续睡眠
- 07:00 → 完全唤醒，整理记忆，恢复正常状态
```

### 发行版可配置
| 配置项 | 默认值 |
|--------|--------|
| 睡眠开始时间 | 00:00 |
| 睡眠时长 | 7小时 |

## 优化建议

1. **精简表达**：用关键词而非完整句子
2. **分层加载**：基础人设+扩展规则分开
3. **定期更新**：根据用户反馈调整OCEAN参数
